# Reference routing

Use progressive disclosure. The files under `examples/` are distributable handwritten style evidence; `temp/pet-examples/pet.md` is not a runtime dependency.

## Routing order

1. Read this index.
2. Read only the relevant rules and language sections.
3. Read the smallest relevant example set, normally one category file.
4. Read `pet-profile.md` only when Dudu's identity matters.
5. Never load all examples for a routine request and never use an example fact as a fact about a new video.

## Task routing

| Request | Read |
| --- | --- |
| General generation or rewrite | [writing-rules.md](writing-rules.md), requested language sections in [language-guides.md](language-guides.md) |
| Translation | Multilingual consistency rules plus source and target sections in [language-guides.md](language-guides.md) |
| Exact artifact or platform format | [output-formats.md](output-formats.md) |
| Dudu identity, gender, names, dates, or profile | [pet-profile.md](pet-profile.md) |
| Health or recovery note | Health sections in [writing-rules.md](writing-rules.md) and [examples/health-and-recovery.md](examples/health-and-recovery.md) |
| AI or fictional story | Generated-content rules and [examples/ai-storytelling.md](examples/ai-storytelling.md) |
| Taxonomy or future corpus maintenance | [taxonomy.md](taxonomy.md) and [source-manifest.md](source-manifest.md) |

## Example routing

- Sleep, blankets, belly-up poses, naps, or relaxation: [examples/sleep-and-relaxation.md](examples/sleep-and-relaxation.md)
- Affection, quiet company, gaze, paws, or kneading: [examples/companionship-and-affection.md](examples/companionship-and-affection.md)
- Play, funny behavior, meowing, attitude, or blank stares: [examples/playful-and-funny.md](examples/playful-and-funny.md)
- Eating, corn, chicken, freeze-dried snacks, or hiccups: [examples/food-and-snacks.md](examples/food-and-snacks.md)
- Bathing, drying, head massage, or routine care: [examples/grooming-and-care.md](examples/grooming-and-care.md)
- Confirmed illness or recovery: [examples/health-and-recovery.md](examples/health-and-recovery.md)
- Growth or before-and-after milestones: [examples/growth-and-milestones.md](examples/growth-and-milestones.md)
- Office cat, home routine, or physical-detail portrait: [examples/daily-life-and-office.md](examples/daily-life-and-office.md)
- Costume, colored light, or music-led styling: [examples/costume-and-music.md](examples/costume-and-music.md)
- Cinematic, fantasy, action, giant-cat, dance, CEO, or phone stories: [examples/ai-storytelling.md](examples/ai-storytelling.md)

Country and location are metadata filters within these files. Only the source supports China/Shanghai/Guilin Park for one real-life example and Japan/Kyoto/Gion for one generated example; do not infer geography for the others.
